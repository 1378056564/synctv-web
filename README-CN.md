## 安装

1. 从`release`下载
2. 放进[synctv](https://github.com/synctv-org/synctv)的`public`文件夹

## 手动构建

```bash
git clone https://github.com/synctv-org/synctv-web
cd synctv-web
npm i
npm run build
```
